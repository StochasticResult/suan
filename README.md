# 玄学排盘助手

一个面向中文用户的网页排盘工具，主打六爻，兼顾八字。

## 功能概览

- 六爻排盘（主功能）
  - 两数字起卦，支持“仅数字”与“数字+时辰”动爻模式
  - 本卦、变卦、互卦、综卦、错卦
  - 周易卦名、卦辞、动爻爻辞
  - 六神、纳甲、六亲、世应、伏神、月建、日辰、旬空（专业盘信息）
  - 一键复制完整盘面文本
- 八字排盘（辅助功能）
  - 四柱、十神、藏干、纳音、旬空
  - 大运 / 流年 / 流月 / 流日联动查看
  - 本地记录存储（localStorage）

## 在线访问

- 已发布到 GitHub Pages https://stochasticresult.github.io/suan/

## 本地运行

这是纯静态网页项目，无需后端：

1. 下载或克隆项目
2. 直接打开 `index.html`

## 技术说明

- 前端：原生 `HTML` / `CSS` / `JavaScript`
- 历法：`lunar-javascript`
- 数据存储：浏览器 `localStorage`（仅八字记录）
